# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.1.x   | :white_check_mark: |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Email security concerns to the project maintainers via the repository contact information.
3. Include a detailed description of the vulnerability, steps to reproduce, and potential impact.
4. You can expect an initial response within 72 hours.
5. We will work with you to understand the issue and coordinate a fix before public disclosure.
