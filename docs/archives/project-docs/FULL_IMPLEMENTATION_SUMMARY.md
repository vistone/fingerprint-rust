**技术架构**:
```python
class DocumentIndexer:  # 文档索引器
    def index_documents(self)  # 索引文档
    def _parse_document(self, file_path)  # 解析文档
    
class DocumentSearcher:  # 文档搜索引擎
    def search(self, query, limit)  # 执行搜索
```
