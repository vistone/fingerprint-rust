1//! # fingerprint-core
2//!
3//! **System-level protection core abstract layer**
4//!
5//! From **single service protection** improve to **system-level protection**, provides system-level core abstractions and interface.
6//!
7//! ## Core Positioning
8//!
9//! `fingerprint-core` is system-level protection core, all external components revolve around this core:
10//!
11//! - **System-level abstractions**: system context, network traffic, protection decision etc.
12//! - **Offense and defense unified interface**: fingerprint abstractions, analysis interface, protection interface etc.
13//! - **Core types and utilities**: type definitions, metadata, utility functions etc.
14//!
15//! ## Core Features
16//!
17//! ### System-level Abstractions
18//!
19//! - **System context** (`SystemContext`): complete network entity information (IP, port, protocol, direction etc.)
20//! - **Network traffic** (`NetworkFlow`): system level network traffic, including context and fingerprint info
21//! - **System protection interface** (`SystemProtector`): system-level protection unified interface
22//! - **System analysis interface** (`SystemAnalyzer`): unified system-level analysis interface
23//!
24//! ### Offense and defense unified abstractions
25//!
26//! - **fingerprint abstractions** (`Fingerprint` trait): support TLS, HTTP, TCP etc.multiple fingerprint types
27//! - **fingerprintmetadata** (`FingerprintMetadata`): including browser, operating system, confidence etc.info
28//! - **TLS fingerprint** (`ClientHelloSignature`): TLS ClientHello signature
29//! - **HTTP fingerprint** (`HttpFingerprint`): HTTP request fingerprint
30//! - **TCP fingerprint** (`TcpFingerprint`): TCP connection fingerprint
31//!
32//! ### core types and utilities
33//!
34//! - **type system**: `BrowserType`, `OperatingSystem` etc.coretype
35//! - **utility functions**: GREASE process, randomly select etc.utility functions
