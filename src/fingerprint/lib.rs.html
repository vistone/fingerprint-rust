1//! # fingerprint
2//!
3//! An independent browser TLS fingerprint library, migrated from golang version.
4//!
5//! ## Features
6//!
7//! - ✅ **Real browser fingerprints**: 90+ real browser fingerprints (Chrome, Firefox, Safari, Opera, Edge)
8//! - ✅ **Real TLS configuration**: Complete TLS Client Hello Spec (cipher suite, elliptic curve, extension, etc.)
9//! - ✅ **JA4 fingerprint generation**: Complete JA4 TLS client fingerprint generation (sorted and unsorted versions)
10//! - ✅ **Fingerprint comparison**: Support fingerprint similarity comparison and best match finding
11//! - ✅ **GREASE processing**: Complete GREASE value filtering and handling
12//! - ✅ **Mobile support**: iOS and Android mobile device fingerprints
13//! - ✅ **HTTP/2 & HTTP/3**: Complete HTTP/2 configuration with HTTP/3 compatibility
14//! - ✅ **User-Agent matching**: Automatic matching of User-Agent strings
15//! - ✅ **Standard HTTP Headers**: Complete standard HTTP request headers
16//! - ✅ **Global language support**: 30+ languages for Accept-Language
17//! - ✅ **Operating system randomization**: Randomly selectable operating systems
18//! - ✅ **High performance**: Zero-allocation on critical paths, concurrent safe
19//! - ✅ **Rust standards**: Strictly follows Rust language standards and best practices
20//! - ✅ **Independent library**: Does not depend on other TLS client libraries
21//! - ✅ **Code quality**: Passes all Clippy checks, follows Rust best practices
